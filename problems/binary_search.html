Binary Search Algorithm Explained

What is Binary Search?

Binary Search is an efficient algorithm to find an element in a sorted array. Think of it like finding a word in a dictionary: you open to a middle page and decide whether to look in the first or second half. Binary Search cuts the search space in half each step, making it super fast!

It relies on the array being sorted to eliminate half of the remaining elements with each comparison.

How Does Binary Search Work?

Here’s the step-by-step process:

1. Start with the entire array. Calculate the middle element.

2. Compare the middle element with the target:

  • If it matches, you’ve found the element!
  • If the target is smaller, search the left half.
  • If the target is larger, search the right half.

3. Repeat until the element is found or the search space is empty.

Key Point: The array must be sorted for Binary Search to work!

Why Use Binary Search?

It’s much faster than Linear Search (checking each element one by one). Binary Search has a time complexity of O(log n), making it ideal for large datasets.

Code Implementation in Java

Here’s a Java implementation of Binary Search that returns the index of the target element or -1 if not found:

public class BinarySearch {
public static int binarySearch(int[] arr, int target) {
int left = 0;
int right = arr.length - 1;

while (left <= right) {
int mid = left + (right - left) / 2;

// Check if target is at mid
if (arr[mid] == target) {
return mid;
}

// If target is greater, ignore left half
if (arr[mid] < target) {
left = mid + 1;
}
// If target is smaller, ignore right half
else {
right = mid - 1;
}
}
return -1; // Target not found
}
}

Code Explanation:

  • left and right track the search range.
  • mid is the middle index, computed as left + (right - left) / 2 to avoid overflow.
  • If the middle element equals the target, return its index.
  • If the target is larger, update left = mid + 1 to search the right half.
  • If the target is smaller, update right = mid - 1 to search the left half.
  • If the loop ends (left > right), return -1 as the element isn’t present.

Examples

Example 1: Finding a Number in a Sorted Array

Array: [2, 3, 4, 10, 40]
Target: 10

Steps:

  • Start: left = 0, right = 4, mid = 2, arr[2] = 4
  • 4 < 10, search right half: left=3, right=4
  • New mid = 3, arr[3] = 10 (found!)
  • Return index 3

Output: Element 10 found at index 3

public class Main {
public static void main(String[] args) {
int[] arr = {2, 3, 4, 10, 40};
int target = 10;
int result = BinarySearch.binarySearch(arr, target);
System.out.println("Element " + target + " found at index " + result);
}
}

Example 2: Element Not in Array

Array: [1, 2, 3, 4, 5]
Target: 6

Steps:

  • Start: left = 0, right = 4, mid = 2, arr[2] = 3
  • 3 < 6, search right half: left=3, right=4
  • New mid = 3, arr[3] = 4
  • 4 < 6, search right half: left=4, right=4
  • New mid = 4, arr[4] = 5
  • 5 < 6, left=5, right=4 (loop ends)
  • Return -1 (not found)

Output: Element 6 not found (-1)

public class Main {
public static void main(String[] args) {
int[] arr = {1, 2, 3, 4, 5};
int target = 6;
int result = BinarySearch.binarySearch(arr, target);
System.out.println("Element " + target + " not found (" + result + ")");
}
}

Example 3: Finding First Element

Array: [10, 20, 30, 40, 50]
Target: 10

Steps:

  • Start: left = 0, right = 4, mid = 2, arr[2] = 30
  • 30 > 10, search left half: left = 0, right = 1
  • New mid = 0, arr[0] = 10 (found!)
  • Return index 0

Output: Element 10 found at index 0

public class Main {
public static void main(String[] args) {
int[] arr = {10, 20, 30, 40, 50};
int target = 10;
int result = BinarySearch.binarySearch(arr, target);
System.out.println("Element " + target + " found at index " + result);
}
}

When to Use Binary Search?

Use Binary Search when:

  • The data is sorted.
  • You need fast searching with O(log n) time complexity.
  • The dataset is large, as it outperforms Linear Search significantly.

Note: If the array isn’t sorted, sorting it first (O(n log n)) is required before applying Binary Search.
