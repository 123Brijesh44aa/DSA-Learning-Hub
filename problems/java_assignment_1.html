Question 1

Animal.java

Method Overriding & Super

Demonstrates method overriding where Dog class overrides the makeSound() method and calls parent's implementation using super keyword.

Animal.java
package javaassignment;

public class Animal {

    void makeSound() {
        System.out.println("animal sound");
    }

}

class Dog extends Animal {

    @Override
    void makeSound() {
        super.makeSound();
    }

}
Question 2

Building.java

Constructor Overloading

Shows constructor overloading and how child class (House) calls parent class (Building) constructors using super().

Building.java
package javaassignment;

public class Building {

    int age;
    String name;

    public Building(int age, String name) {
        this.age = age;
        this.name = name;
    }

    public Building(int age) {
        this.age = age;
        this.name = "Anonymous";
    }
}

class House extends Building {

    int address;

    public House(int age, String name, int address) {
        super(age, name);
        this.address = address;
    }

    public House(int age) {
        super(age);
        this.address = address;
    }

}

Question 6

Parent.java

Method Overriding & Super

Demonstrates inheritance, constructor chaining, method overriding, and the use of 'this' and 'super' keywords.

Parent.java
package javaassignment;

public class Parent {

    void display() {
        System.out.println("parent display");
    }

    public Parent() {
        System.out.println("this is parent class constructor");
    }
}

class Child extends Parent {

    @Override
    void display() {
        System.out.println("child display");
    }

    Child() {
        super();
        System.out.println("child class constructor");
    }

    void show() {
        this.display();
        super.display();
    }

}

Person.java

Field Shadowing

Shows field shadowing where Employee class has its own age and name fields that shadow the parent's fields.

Person.java
package javaassignment;

public class Person {

    int age;
    String name;

    public Person(int age, String name) {
        this.name = name;
        this.age = age;
    }
}

class Employee extends Person {

    int age;
    String name;
    double salary;

    public Employee(int age, String name, double salary) {
        super(age, name);
        this.salary = salary;
    }

}
